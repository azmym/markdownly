# Security Policy

## Supported versions

markdownly is a solo project without long-term support branches. Only the most recent release is supported; please update before reporting an issue.

| Version        | Supported |
| -------------- | :-------: |
| Latest release |    Yes    |
| Older releases |    No     |

## Reporting a vulnerability

Please do not open a public issue for security problems. Instead, open a private report using GitHub Security Advisories:

[https://github.com/azmym/markdownly/security/advisories/new](https://github.com/azmym/markdownly/security/advisories/new)

Include a description of the issue, reproduction steps, the affected version, and any suggested mitigation.

## Response time

This is a best-effort project. Expect an initial acknowledgement within about a week. Fix timelines depend on severity and available time.
